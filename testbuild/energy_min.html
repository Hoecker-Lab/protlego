Energy minimization¶

It is also possible to minimize the energy of the chimeras generated using protlego. Protlego has in-built choices between “charmm” and “amber” forcefield to minimize the prtoein models. The code below shows how this function can be utilised.

import sys
sys.path.append("YOUR_PATH_TO_PROTLEGO_FOLDER")

from protlego.all import *

User can run mimization by using the code as shown below. The function requires the name of the chimera object to be minimized, the choice of forcefield, i.e., “amber/charmm”, choice of using the GPU and path to the output file.

[5]:

scorer.mm_energies.minimize_potential_energy(chimera,"charmm", cuda=False, output='.')
